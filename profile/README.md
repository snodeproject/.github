## 👋 Benvenuto in Snode

Snode è un progetto open source che punta a portare un'esperienza simile a Node.js in un runtime sviluppato interamente in Python.

Il progetto nasce con l'obiettivo di essere semplice da comprendere, facile da estendere e divertente da utilizzare, sia per imparare come funziona un runtime JavaScript sia per sperimentare nuove idee.
